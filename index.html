Windows Print Instructions

CRITICAL: Even though you are feeding a small PVC card, you MUST tell the printer it is a 10x15cm sheet. Our app will magically center the image for you.

  • Set Printer to Epson L3110 Series.
  • Set Paper Size to 10x15 cm (4x6 in). Do NOT select your custom card size.
  • Set Margins to None / Borderless.
  • Set Scale to 100% (No "Fit to paper").
  • Set Paper Type to Epson Matte or Premium Glossy.
  • Set Quality to High or Photo.
  • Place the PVC card directly into the rear feed tray (centered).
