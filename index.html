Dragon Sheet Print Instructions

You are printing to a 10x15cm Dragon Sheet. Please ensure these settings are selected in the Windows Print Dialog:

  • Set Printer to your Epson Inkjet (L3110/L3250).
  • Set Paper Size to 10x15 cm (4x6 in).
  • Check the "Borderless" option (Crucial so printer doesn't shrink it).
  • Set Scale to 100% (No "Fit to paper").
  • Set Paper Type to Epson Matte (or Premium Glossy).
  • Set Quality to High or Photo.
  • Load the Dragon Sheet film and click Print!
